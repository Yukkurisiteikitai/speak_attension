# 0007: マップ復帰導線と共有画面向け会議UI

Date: 2026-07-21  
Status: accepted

## 決定

すべてのマップに、全ノードを画面内へ戻す日本語ラベル付きの「全体を表示」を常設する。パンは自由のまま維持し、通常発言によるノード追加では閲覧位置を自動変更しない。

会議ライブマップは中央ルートから議題を左右へバランス配置し、発言を各議題の外側へ時系列に並べる。会議画面の右レールは「進行」「分析」のタブへ分け、手入力・リプレイ・発話ログは初期状態で閉じた入力ドックへまとめる。

## 背景

自由なパン操作でマップ全体を画面外に見失うと、アイコンだけの標準操作から復帰方法を判断しにくかった。会議ライブマップと補助パネルも縦方向へ伸び、画面共有時に主情報であるマップを圧迫していた。

## 影響

- 自動全体表示は初回、アイデアのフェーズ切替、会議整理の再生成に限定する。
- 会議レイアウトはノード高と発言量から左右の使用高を決定的に均す。
- 利用形態はファシリテーター1人の操作と画面共有を前提とする。
- LAN共同編集、クラウド公開、認証、DB保存は追加しない。
