# 0003: ローカル LLM は LM Studio に限定する

Date: 2026-07-09  
Status: accepted

## 決定

ローカル LLM を任意の補助処理に利用する。接続先とフォールバックを含む現在の制約は [AGENTS.md](../../AGENTS.md) を正とする。

## 背景

ルールベースの検出結果について、外部サービスにデータを渡さずに追加の観点を得る必要があった。同時に、ローカルモデルが停止・未設定でもアプリが使える必要がある。

## 影響

- `llmClient` は OpenAI 互換のローカルエンドポイントだけを扱う。
- レポートレビューとアイデアのグループ化は、失敗時に決定的なルールベース処理へ戻す。
- LLM の応答は既存のルールベース結果を破棄する根拠にはしない。
