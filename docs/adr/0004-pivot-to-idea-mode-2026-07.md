# 0004: アイデア出しモードを主モードにする

Date: 2026-07-09  
Status: accepted

## 決定

アプリの既定画面をアイデア出しモードにし、会議ダッシュボードは副モードとして残す。アイデア出しでは、発話からキーワードを収集し、放射状表示からグループ化後のマインドマップへ遷移させる。

## 背景

アイデアを広げ、採用候補を整理し、出典となる発話とともに持ち出す体験を優先することにした。会議モードの過去の「放射状レイアウトを採用しない」という判断は、このモードには適用されない。

## 影響

- `App.tsx` はアイデア出しモードを既定で開く。
- `ideaSession`、`ideaExtraction`、`ideaGrouping`、`ideaLayout`、`IdeaModeView` が主経路になる。
- キーワードから発話へのリンクを保った Markdown/JSON エクスポートを提供する。
