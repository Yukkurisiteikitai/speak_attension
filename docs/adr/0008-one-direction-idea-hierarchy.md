# 0008: アイデア整理マップを一方向の階層にする

Date: 2026-07-21  
Status: accepted

## 決定

グループ化後のアイデア整理マップは、左から右へ「テーマ → グループ → キーワード」と進む一方向の階層で表示する。同じグループのキーワードは連続した縦ブロックにまとめ、接続線も右向きに統一する。

## 背景

テーマからグループを左右へ交互に配置すると、画面中央をまたぐ主線によって異なるグループが一続きの関係に見えやすかった。利用者はテーマからグループを選び、その中の案を追うため、読む方向と情報の階層を一致させる必要がある。

## 影響

- グループ化前の発散マップは従来の放射状表示を維持する。
- グループ化後は全グループをテーマの右側へ配置する。
- 将来キーワードから子案を伸ばす場合も、同じ右向きの階層を継続できる。
- 長短ラベルと複数グループの非重複、およびグループ順をレイアウトテストで保護する。
