# attention_mindmap

## 概要

複数人での議論を見える化し、会議の進行を支援するローカルWebアプリです。会話から話題を抽出してマインドマップに表示し、話し忘れや未解決の論点を把握しやすくします。

## 機能

- 会話から主要なトピックを抽出して可視化
- 議題の解決・未解決や、決定事項を管理
- 抜け漏れになりそうな論点を表示
- 日本語音声認識と手入力に対応
- アイデア出し用のキーワード整理・エクスポート機能
- ~~意見の相違を構造化して表示~~

## 起動方法

Node.js を用意したうえで、以下を実行します。

```sh
npm install
npm run dev
```

ブラウザで `http://127.0.0.1:5173/` を開いてください。音声入力を使う場合は、マイクの利用を許可してください。
