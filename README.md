# attention_mindmap

## 概要

話した内容からアイデアや議題を見える化する、ローカルWebアプリです。主モードではアイデアを集めて整理し、副モードでは会議の進行と抜け漏れ確認を支援します。

## 機能

- 日本語の音声または手入力からアイデアのキーワードを収集
- キーワードのグループ化、採用・保留・却下、Markdown / JSON出力
- 会話を「話題 → 課題 → 原因 → アクション／別案」の意味階層としてリアルタイムにマップ表示
- 会議後の課題、決定、担当、期限、未解決事項を根拠発言付きで整理
- 会議で見つけた課題から、そのままアイデア出しを開始
- すべてのマップに、画面内へ戻す「全体を表示」操作を用意

## 利用想定

ファシリテーター1人がブラウザを操作し、参加者には画面共有や外部ディスプレイで見せる使い方を想定しています。複数端末からの同時編集には対応していません。

会議状態はブラウザ内で扱います。任意のLLM補助機能はローカルのLM Studioだけを利用し、未設定または接続失敗時もルールベース処理で全機能を利用できます。音声認識はブラウザのWeb Speech APIに依存するため、音声データの扱いは利用ブラウザの仕様を確認してください。

## 起動方法

Node.js を用意したうえで、以下を実行します。

```sh
npm install
npm run dev
```

ブラウザで `http://127.0.0.1:5173/` を開いてください。音声入力を使う場合は、マイクの利用を許可してください。

クラウド公開、ユーザー認証、DB保存は行いません。セッション状態はリロードすると失われるため、必要な結果はMarkdownまたはJSONで保存してください。
